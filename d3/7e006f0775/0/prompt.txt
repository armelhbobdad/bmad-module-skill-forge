## 2. docs/concepts.md — New File

Plain-English definitions, 2-4 sentences each, with concrete examples. This is the bridge between "Getting Started" and "How It Works."

Sections:
- **Agent Skills** — instruction files that tell AI agents how to use your code
- **Provenance** — every instruction links to the exact source file and line
- **Confidence Tiers (T1/T2/T3)** — how much you can trust each piece of info
- **Capability Tiers (Quick/Forge/Deep)** — what tools you have determines your level
- **Drift** — when code changes but skill instructions haven't been updated
- **Version Pinning** — skills remember which code version they were built from
- **BMAD Module** — SKF is a plugin for the BMad workflow framework
- **Ferris** — the AI agent persona that runs SKF workflows
- **Zero Hallucination** — if it can't trace to source code, it doesn't get included
